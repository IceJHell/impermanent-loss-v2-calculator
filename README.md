# Калькулятор непостоянных потерь V2

Статический калькулятор impermanent loss для пулов V2 на примере ETH/USDC и BTC/USDC.

## Что умеет

- считает распределение позиции 50/50;
- показывает результат LP против HODL;
- подтягивает справочную текущую цену ETH/BTC через CoinGecko;
- добавляет оценку комиссий по сроку позиции и годовой доходности;
- содержит памятку по типовым движениям цены.

## Как открыть

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 4173 --bind 127.0.0.1
```

После этого страница будет доступна по адресу:

```text
http://127.0.0.1:4173/
```
